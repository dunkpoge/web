# Dunk Poge — Web

This repository contains the **compiled production build** of the Dunk Poge frontend. It serves [dunkpoge.com](https://dunkpoge.com) via GitHub Pages.

> ⚠️ Do not edit files in this repo directly. This is a build output folder.

## Want to run or modify the frontend?

The source code lives here:  
👉 [github.com/dunkpoge/frontend](https://github.com/dunkpoge/frontend)

```bash
git clone https://github.com/dunkpoge/frontend
cd frontend
yarn install
yarn start
```

## Deploying

To update this repo, build from the frontend source and push the output:

```bash
yarn build
# then copy build/ contents here and push
```
